## 2. コアコンセプト：「The Staking Incantation（ステーキング詠唱）」

#### 各パネル詳細

**Panel 1「構え」(0.0-1.0s)**
- ミディアムショット。暗い宇宙空間的な虚空
- TORICOちゃん（リアル調コスプレ風）が右手を胸の青い宝石ペンダントに当てる
- 目は閉じている。宝石から淡い青い光が脈動
- 背景：深いインディゴの虚空に微かな六角形グリッド（ブロックチェーン風）
- カメラ：静止、正面ミディアム

**Panel 2「覚醒」(1.0-2.5s)**
- 顔と上半身のクローズアップ
- 目がパッと開く — 鮮やかな青、発光
- 唇が開き詠唱を開始。ペンダントから青いエネルギーの糸が上方に立ち昇る
- 白銀の髪が浮き上がり青い光を受ける
- カメラ：ゆっくりプッシュイン

**Panel 3「ネットワーク展開」(2.5-4.0s)**
- ワイドショット。両腕を大きく広げる
- 青いエネルギーの糸が全方向に伸びる
- 球状配置の8-12個の小さな八面体クリスタル（バリデータノード）に到達
- ノード同士が光の線で接続 = バリデータネットワーク/魔法陣
- 各ノード内にETHダイヤ形が微光
- カメラ：引きのワイドショット

**Panel 4「叫び」(4.0-5.0s)**
- ミディアムショット、やや煽りアングル（ヒロイック）
- 口を大きく開けて **「イーサリアム！」** と叫ぶ
- 赤いドレスvs青いエネルギーの鮮烈なコントラスト
- ノードが順番にフラッシュ = コンセンサスメカニズムが呪文を承認
- 光の波紋がネットワークを伝播
- カメラ：やや煽りのミディアム

**Panel 5「収束」(5.0-6.5s)**
- ワイドショット
- 全ノードが上方の一点に集束開始
- 光る文字/ハッシュ記号のようなデータパーティクルが接続線を流れる
- 両手の上に小さく強烈に輝くETH八面体が結晶化開始
- マントがエネルギーの上昇気流で劇的にはためく
- カメラ：ゆっくりオービタル（回り込み）

**Panel 6「鍛造」(6.5-7.5s)**
- 八面体のクローズアップ。胴体サイズに成長

- ETHダイヤモンドの幾何学模様が表面に
- 白い手袋の両手が下からフレーミング
- カメラ：クローズアップ、浅い被写界深度

**Panel 7「The Merge」(7.5-8.5s) ★クライマックス**
- 八面体が一瞬で二つに分裂

  - 青半分（PoS / ETHトレジャリー）
- 激しく再融合 → 眩い白色閃光
- 衝撃波リングが外側に拡大
- カメラ：ワイドショット、衝撃波で揺れ

**Panel 8「世界変容」(8.5-9.5s)**

- 完成した八面体が頭上を冠のように周回
- カメラ：ドリーアウトのワイドショット

**Panel 9「アイコン」(9.5-10.0s)**
- 最終ヒーローポーズ
- 片手をカメラに向けて伸ばす（自信に満ちた表情）

## 7. 技術最適化メモ

- **1ビート1動詞**: 各タイムセグメントに主アクション1つ → 動き破綻防止
- **カメラ明示指定**: push-in → wide → orbit → static → 各ビートで明記
- **色彩制限**: 青/インディゴ支配 + Merge時のみオレンジ → インパクト最大化
- **動きは穏やかに**: Seedance 2.0は穏やかな動きの方が品質が高い
- **音声キュー**: orchestral swell、crystalline chimes、deep boom → ネイティブ音声生成活用
- **入力画像は2K以上**: ぼやけた参照 = ぼやけた出力
- **リアル調の注意点**: 不気味の谷を避けるため、魔法エフェクト発動後はVFX的光で顔を部分的に隠す構図も有効
